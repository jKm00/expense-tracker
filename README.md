# Expense Tracker

A simple expense tracking application. View it live at [ex.edvardsen.dev](https://ex.edvardsen.dev).

![Home Page](docs/home-page.jpg)

## Development

1. Copy the example environment file:
   ```bash
   cp .env.example .env
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```
